# Aplikasi Kasir Swalayan (POS)

Aplikasi kasir berbasis web sederhana yang dibangun untuk memenuhi tugas kuliah / project pribadi.

## Fitur Utama
* ✅ Login Admin
* ✅ Transaksi Kasir (Scan Barcode & Search)
* ✅ Cetak Struk Belanja
* ✅ Laporan Keuangan (Omzet & Laba)
* ✅ Export Laporan ke Excel
* ✅ Manajemen Stok (Masuk/Keluar)

## Teknologi
* PHP Native
* MySQL Database
* Bootstrap 5
* CSS Custom

## Cara Install
1. Download Source Code ini.
2. Import file database `kasir_toko.sql` ke phpMyAdmin.
3. Atur koneksi database di file `koneksi.php`.
4. Buka di browser.
   * **User:** admin
   * **Pass:** admin
